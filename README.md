# Yemek Sitesi DB First Projesi

Bu proje, ASP.NET Core MVC ve Entity Framework Core DB First yaklaşımıyla geliştirilmiş bir yemek sitesi uygulamasıdır. SQL Server üzerinde oluşturulan tablolar Entity Framework ile projeye aktarılmış ve MVC yapısı üzerinde kullanıcı/admin işlemleri geliştirilmiştir.

## Kullanılan Teknolojiler

- ASP.NET Core MVC
- C#
- Entity Framework Core
- DB First
- SQL Server
- Scaffold-DbContext
- Bootstrap
- HTML / CSS / JavaScript

## Proje Özellikleri

- DB First yaklaşımı
- SQL Server tablolarından model üretimi
- Kullanıcı arayüzü
- Admin paneli
- Kategori yönetimi
- Yemek listeleme
- CRUD işlemleri
- Mesaj yönetimi
- Raporlama sayfaları
- PDF / Excel çıktı alma mantığı

## Proje Amacı

Bu proje, hazır bir veritabanı üzerinden Entity Framework Core DB First yaklaşımıyla uygulama geliştirme pratiği yapmak amacıyla hazırlanmıştır.

## Teknik Kazanımlar

- Scaffold-DbContext kullanımı
- SQL Server tablolarından model oluşturma
- MVC controller ve view yapısı
- Admin panel geliştirme
- Kullanıcı ve yönetici tarafının ayrılması
- Veri listeleme ve raporlama mantığı
